## Features
- jpeg image frame 
    - Upload in the git page 
    - Conversion within the git page
    - Flash upload to the device

- Buttons for toggling between images 

- Long Press to change mode 
    1. Time display mode
    2. Gallery display mode
    3. Count down timer (25:00 Pomodoro)

## Hardware
- Display: ST7789 1.54" IPS 240x240
    - SPI connection

    - input user local SSID and PW
